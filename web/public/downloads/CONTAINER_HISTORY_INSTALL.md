# ELS 컨테이너 이력조회 — 설치 설명서

데스크탑(Windows)과 안드로이드에서 **컨테이너 이력조회**를 쓰려면 아래 설치 파일을 다운로드한 뒤 설치하면 됩니다.

---

## 1. Windows (데스크탑)

### 1-1. 다운로드

- **컨테이너 이력조회** 페이지 상단 **다운로드 및 설치** 섹션에서  
  **Windows 설치 프로그램 (.exe)** 를 클릭해 다운로드합니다.

### 1-2. 설치 요구사항

- **Python** (PC에 설치, PATH 등록)
- **Chrome** 브라우저

### 1-3. 설치 방법

1. 다운로드한 `els-container-history-setup.exe` (또는 표시된 파일명)를 실행합니다.
2. 설치 경로를 선택한 뒤 **다음**으로 진행해 설치를 마칩니다.
3. 설치가 끝나면 **ELS 컨테이너 이력조회** 앱을 실행합니다.
4. 앱에서 로그인 → 컨테이너 입력/업로드 → 조회·엑셀 다운로드를 사용할 수 있습니다.

### 1-4. 사용 방법

- 앱 실행 후 화면이 열리면, 이 웹 페이지와 동일하게 **로그인** → **엑셀 업로드 또는 번호 입력** → **조회** → **엑셀 다운로드** 순서로 사용하면 됩니다.

---

## 2. Android (모바일)

### 2-1. 다운로드

- **컨테이너 이력조회** 페이지 상단 **다운로드 및 설치** 섹션에서  
  **Android 앱 (.apk)** 를 클릭해 다운로드합니다.
- 모바일에서 페이지에 접속했다면, 같은 페이지에서 **Android 앱 (.apk)** 를 눌러 받을 수 있습니다.

### 2-2. 설치 방법

1. 다운로드한 `.apk` 파일을 열어 설치합니다.
2. **출처를 알 수 없는 앱** 설치 허용이 필요하면, 설정에서 허용 후 다시 설치합니다.
3. 설치가 끝나면 **ELS 컨테이너 이력조회** 앱을 실행합니다.

### 2-3. 사용 방법 (같은 Wi‑Fi 필요)

- Android 앱은 **데스크탑 앱을 서버로** 사용합니다.
1. **PC에서** ELS 컨테이너 이력조회 **데스크탑 앱**을 실행합니다.
2. PC와 Android 기기가 **같은 Wi‑Fi**에 연결되어 있는지 확인합니다.
3. Android 앱을 실행한 뒤, 화면에 안내된 대로 **PC의 IP 주소와 포트**(예: `192.168.0.10:2929`)를 입력합니다.
4. 연결되면 컨테이너 이력조회 화면이 열리며, 데스크탑과 동일하게 조회·다운로드를 사용할 수 있습니다.

---

## 3. 요약

| 구분 | 다운로드 | 설치 후 |
|------|----------|---------|
| **Windows** | .exe 설치 프로그램 | PC에서 바로 로그인·조회·다운로드 사용 |
| **Android** | .apk 앱 | 같은 Wi‑Fi에서 데스크탑 앱 실행 후, 앱에서 PC 주소 입력해 접속 |

- **exe, apk** 파일은 **컨테이너 이력조회** 페이지의 **다운로드 및 설치** 섹션에서 링크로 호출해 받을 수 있습니다.
- 문의 사항은 사내 담당자에게 요청하시면 됩니다.
