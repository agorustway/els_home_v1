// turbo-all

1. Build the Next.js project
```powershell
npm run build
```

2. Stage all changes
```powershell
git add .
```

3. Commit changes
```powershell
git commit -m "chore: 자동 배포 업데이트"
```

4. Push to remote repository
```powershell
git push
```
