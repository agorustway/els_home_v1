1. 프로젝트 현황판 읽기
```
docs/MISSION_CONTROL.md 파일을 읽어서 현재 프로젝트 상태를 파악한다.
```

2. 협업 규칙 확인
```
docs/RULES.md 파일을 읽어서 형과의 협업 규칙을 숙지한다.
```

3. (선택) 필요한 경우 추가 참조
```
- 개발 이력이 필요하면: docs/DEV_LOG.md
- 아키텍처 이해가 필요하면: docs/ARCHITECTURE.md
- 운영/배포 관련이면: docs/RUNBOOK.md
```
