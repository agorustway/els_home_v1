1. 프로젝트 및 앱 현황판 읽기
```
docs/01_MISSION_CONTROL.md 와 docs/02_APP_CONTROL.md 파일을 읽어서 웹과 안드로이드 앱의 전체 진척도와 현재 집중 목표를 즉각 파악한다.
```

2. 협업 규칙 확인
```
docs/03_RULES.md 파일을 읽어서 형과의 협업 규칙을 숙지한다.
```

3. (선택) 필요한 경우 추가 참조
```
- 시스템 통신/설계 확인: docs/04_ARCHITECTURE.md
- UI/UX 다크 모드 등 디자인 정책 확인: docs/05_DESIGN_SYSTEM.md 
- 최근 개발 롤백/히스토리가 필요하면: docs/06_DEV_LOG.md
- 운영/배포 관련 이슈면: docs/07_RUNBOOK.md
```
