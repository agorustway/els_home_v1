5. **기록 수정 불가**: 읽기 전용 모달이었음 → 수정 가능한 폼(컨테이너, 씰, 타입, 메모)으로 전면 개편, `saveHistoryEdit()` 함수 추가.
6. **사진 업로드 반복 실패**: `trip_id`가 빈 문자열로 전송되어 서버 DB 조회 실패 → `lastTripId` 보존으로 종료 후에도 업로드 가능.

- 입력 폼: [컨테이너+사이즈] / [씰번호+종류] 2행 구성으로 잘림 방지
