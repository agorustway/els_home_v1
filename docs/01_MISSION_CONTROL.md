### 🚀 현재 상태 (CURRENT STATUS)
- **버전**: v5.12.1
- **주요 사항**:
  - **[AI 배차 통계 무결성]**: 서버에서 요약문을 만들 때 화주별(글로비스/모비스) 합산이 누락되지 않도록 로직 보완. AI는 이제 직접 계산할 필요 없이 서버가 준 최종 수치만 읽으면 됨.
  - **[GPS 신뢰도]**: 앱 포그라운드 복귀 시 불필요한 GPS 재기동 제거 및 필터 임계값 조정을 통한 경로 정밀도 향상.
  - **[배차 RAG 고도화]**: 아산 배차판의 오더(계) vs 실제 배차수량 구분, 지역 컬럼(부산/인천 등) 파싱, 셀 메모 시간 해석 규칙을 AI 지시사항에 완벽 반영.
  - **[AI 배차 분석]**: 날짜별 소계(byDate) 집계 추가로 "어제/오늘/특정일" 배차 대수를 명확히 구분하여 답변.
  - **[AI 챗봇 고도화]**: 차량 출발지/도착지 주소 추적 및 내일 기상 예보 데이터 주입. Omni-Intelligence 지침 적용으로 봇의 기술적 추론 및 통찰 제시 능력 강화.
  - **[웹 관제 UX 개선]**: 전체화면 시 '전체화면 해제' 및 '포커스 맞추기' 버튼을 중앙 필터 바 내부로 통합하여 버튼 겹침 및 레이아웃 붕괴 해결.
  - **[네비게이션 모드]**: 실시간 추적 시 지도가 차량을 부드럽게 따라가는 `panTo` 방식 (웹/앱 공통)
  - **[앱 지도 제어]**: 중복 버튼 통합(📍 내 위치), 마커 클릭 시 상세 경로 정보 패널 노출. 자동 추적 일시 중지 로직 포함.
  - **[앱 애니메이션]**: GPS 갱신 시 마커가 부드럽게 활주하는 `animateMarker` 적용
  - **[UI 정밀 조정]**: 차량 상세 모달(Full-width), 활동 로그(200px), 운전원 관리(nowrap) 최적화 완료.
  - **[GPS 신뢰도]**: 지하주차장 종료 시 GPS 타임아웃 대응 (고정밀 4초 -> 저정밀 2초 재시도 -> 최종 fallback 강제 기록).
  - **[필터 보정]**: `marker_type`이 포함된 포인트는 이상치 필터(`trimEndpointOutliers`)에서 제외하여 경로 보존.
