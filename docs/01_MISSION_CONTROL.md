# ELS MISSION CONTROL v3.5.0
> 마지막 업데이트: 2026-03-22 17:35

## 🎯 현재 목표
드라이버 앱 v3.5.0 빌드 & 실기기 테스트 (오버레이 로딩/사진 업로드 수정 반영)

## 📋 최근 변경 (v3.5.0)
- **오버레이 플러그인 로딩 개선**: `getOverlay()` 지연 로딩 도입. 초기 로드 시 `null`로 인한 "안드로이드 전용" 에러 해결.
- **사진 업로드 서버 안정화**: 서버단 인증(Auth) 의존성 제거, `maybeSingle()` 적용으로 초기 운행 기록 조회 시 오류 방지.
- **UI/UX 개선**: 하단 탭 내비게이션에 **종료** 버튼 즉시 배치 (빨간색).
- **데이터 바인딩 수정**: 기사 정보 조회 시 **차량코드(기존 D-ID)** 누락 및 유효성 검사 로직 완화.
- **문구 수정**: "오버레이(선택)"에서 (선택) 문구 제거, D-ID -> 차량코드로 변경.

## 📋 최근 변경 (v3.4.0)
- **오버레이 완전 재구현**: `OverlayPlugin` 전면 개편(startActivityForResult).
- **GPS 관제 수집**: 속도별 가변 주기 (30/60/120초) 및 상태별 마킹(PAUSE/STOP).
- **기록 상세 모달**: 사진 조회/추가 업로드/삭제 기능.
