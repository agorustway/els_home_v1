## 📋 최근 변경 (v4.2.11 — 2026-03-29 11:35)
- **디자인 시스템 통일**: 팝업 모달(운행전점검, 약관 등)의 모서리 라운딩(`border-radius`)을 버튼과 동일한 `4px`(`--radius`)로 통일하여 정돈된 톤앤매너 확보.
- **설정 페이지 고도화**: 운전원 정보 조회 시 운전원, 차량, 샤시 사진을 확인할 수 있는 3분할 프로필박스 UI 추가. (DB 컬럼 `photo_driver`, `photo_vehicle`, `photo_chassis` 연동 준비 완료)
- **DB 마이그레이션**: 사진 컬럼 추가를 위한 SQL 스크립트(`docs/migration_154_photos.sql`) 작성.

## 📋 최근 변경 (v4.2.10 — 2026-03-29 11:24)
- **운행 전 필수점검 도입**: 운사업법 시행규칙(별지 14-5) 기반의 5대 핵심 체크리스트 팝업 추가. 미점검 시 운행 시작 불가하도록 차단.
- **점검 데이터 DB 연동**: 5개 점검 항목(`chk_brake`, `chk_tire`, `chk_lamp`, `chk_cargo`, `chk_driver`) 상태를 `/api/vehicle-tracking/trips` 로 전송 및 업데이트 연동 완료. (관련 마이그레이션 문서 `docs/` 저장 완료)
- **앱 화면 세로 고정**: `AndroidManifest.xml`에 `screenOrientation="portrait"` 강제 적용.
- **공지사항 필터 탭**: 카테고리별(안전교육, 작업안내, 긴급알림 등)로 열람 가능한 가로 스크롤 탭 버튼 추가. (해당 카테고리 DB 필드 추가 스크립트 제공 완료)
- **긴급알림 중복 방지**: 로컬스토리지(Store)에 `emergencyIds`를 영구 저장하여 앱 재시작 시 동일 긴급알림 무한 팝업 문제 완벽 해결.
- **일지 조회 편의 개선**: 검색 필드 자리 차지 문제를 해결하기 위해 placeholder를 '날짜조회', '월별조회'로 심플하게 축약.

## 📋 최근 변경 (v4.1.75 — 2026-03-28 23:57)
- **헤더 뒤로가기 버튼 제거**: 상세 페이지 진입 시 로고가 오른쪽으로 밀리는 현상을 방지하기 위해 헤더의 뒤로가기 버튼을 삭제함. (상세 페이지 내부에 이미 전용 뒤로가기 버튼이 존재함)
- **공지 상세 버튼 최적화**: 공지사항 상세 상단의 '목록으로' 버튼이 가로로 너무 길게 늘어지던 문제를 해결하여 일지 상세와 동일하게 짧은 버튼으로 수정함.
- **레이아웃 정밀 조정**: 좁은 모바일 화면에서도 헤더의 차량번호와 상태 뱃지가 두 줄로 겹치거나 밀리지 않도록 고정 조치함.
- **헤더 및 로고 정상화**: 이미지 경로 오류(`/images/` 폴더 누락)를 해결하여 GIF 로고가 정상적으로 표시되도록 수정함.
- **레이아웃 일관성 강화**: 설정 화면, 공지 상세, 일지 상세 페이지의 헤더 높이 및 좌우 여백을 메인 화면과 동일하게 맞춰 UX 통일성을 확보함.
- **공지 본문 가독성 개선**: 본문 텍스트의 줄바꿈(`\n` -> `<br>`) 지원 및 톤앤매너(Noto Sans KR)를 여타 페이지와 통일함.
- **일지 정보 보완**: 예전 운행 기록에서 `ended_at`이 비어있는 경우 `completed_at`을 대체 표시하여 모든 완료 건에서 종료 일시가 확인 가능하도록 개선함.

## 📋 최근 변경 (v4.1.64 — 2026-03-28 22:50)
- **일지 상세 가독성 강화**: 운행 시작 및 종료 일시를 분리 표기하여 기사님들이 한눈에 확인 가능하도록 개선함.
- **온보딩 가이드 고도화**: 최초 권한 설정을 마친 후 차량 정보(차량번호, 이름 등)가 없을 경우 자동으로 설정 페이지로 이동시키는 안내 로직 구현.
- **버전 정합성 확보**: 앱 내부 설정 화면 및 인트라넷 내 노출되는 버전을 `v4.1.64`로 통일하고 캐시 갱신 문제 해결.

## 📋 최근 변경 (v4.1.46)
- **UI & 레이아웃 개편**: 헤더 상태뱃지 옆으로 설정(⚙️) 버튼을 빼내어 화면 활용도를 높임. 운행 상단바 디자인을 깔끔한 연한 배경의 카드로 개선함.
- **오버레이 초기 숨김**: 운행 시작 버튼을 눌렀을 때 앱 위에 즉시 위젯이 나타났던 문제를 수정하여 최초 가시성을 GONE으로 변경함. (앱을 최소화할 때만 나타나는 기획 의도 실현)
- **권한 연동 버튼 색상화**: 권한 설정 창에서 필수 권한 미달 시 붉은 핏빛 경고 펄스(맥박) 파동 CSS 적용. 허용 시 파란색(Primary) 버튼으로 정상 동기화 됨.
- **오기입 문구 직관화**: 컨테이너 유효성 검사 실패 시 "체크디짓 미일치"를 "컨테이너번호 오기입 넘버를 확인해주세요" 로 친절하게 변경.

## 📋 최근 변경 (v4.1.44)
- **운송 종료 일시 표시**: 운행 시작 후 종료를 누르면 모든 정보가 지워지더라도, 상단에 `[운송종료: 2026-03-28 15:30]` 형태로 유지되도록 캐싱 구현.
- **캐시 삭제 메뉴**: 앱 초기화 메뉴 옆에 Session Storage/Cache만 비우고 새로고침하는 캐시 지우기 메뉴 추가.
