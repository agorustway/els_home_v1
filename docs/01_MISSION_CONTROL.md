## 📋 최근 변경 (v4.1.42 — 2026-03-28 16:38)
- **오버레이 고도화**: 
  - 위젯 크기를 1.5배 키우고, 초기 위치를 우측 상단(네비게이션바 여백 고려)으로 조정.
  - **앱 최소화 연동**: 앱이 포그라운드일 때는 숨기고, 홈 화면으로 나가거나 최소화 되었을 때만 오버레이가 자동으로 나타나도록 `onResume/onPause` 연동 완료.
- **앱내 상태바 구현**: 운행 탭 상단에 **[시간 | GPS 간격 | 현위치 주소]**가 실시간으로 업데이트되는 상태바 추가.
- **배포 자동화**: Build 88 APK 생성 및 Vercel/GitHub Push 완료.

## 📋 최근 변경 (v4.1.40 — 2026-03-28)
- **원본 기능 100% 복구**: 날아갔던 운행 일지 페이지, 기사 정보 조회 버튼, 모든 컨테이너 종류(TANK, FLAT RACK 등) 완벽 복원.
- **로고 복원**: 헤더에 `els_logo_full.gif` 이미지가 정상 출력되도록 경로 수정.

## 📋 최근 변경 (v4.1.39 — 2026-03-28)
