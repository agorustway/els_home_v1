# ELS Solution - Employee Portal

ELS Solution 임직원 포털 및 공식 웹사이트

## 최신 업데이트 (v35)
- NAS 자료실: 다중 선택 및 ZIP 압축 다운로드 기능 추가
- NAS 자료실: 드래그 앤 드롭으로 파일 이동 및 업로드
- NAS 자료실: Long Press로 다중 선택 모드 진입
- 헤더 메뉴: 간격 확대 (15px → 28px)
- 헤더 메뉴: 로그인 버튼을 텍스트 링크로 변경
- 파일 다운로드: 원본 파일명 보존 및 MIME 타입 개선

## 기술 스택
- Next.js 14
- Supabase (인증 및 데이터베이스)
- WebDAV (NAS 연동)
- React

## 주요 기능
- 임직원 전용 자료실 (NAS)
- 업무 보고 시스템
- 게시판 시스템
- 지점별 서비스
- 관리자 대시보드
